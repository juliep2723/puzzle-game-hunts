# The Blockbuster Game Rubric: Top Concepts

**The Criteria:**
*   **Smart Silly:** Funny premise, but treated with "Blockbuster" seriousness (e.g., *Men in Black*, *The LEGO Movie*, *Spy Kids*).
*   **Accessible Stakes:** Problems that are visually obvious or emotionally resonant.
*   **The 6-12-Adult Test:** A 6-year-old can follow the action, a 12-year-old feels respected, an adult catches the deeper themes.
