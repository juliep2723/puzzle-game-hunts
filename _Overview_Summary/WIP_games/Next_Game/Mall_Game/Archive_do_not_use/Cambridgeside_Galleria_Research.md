### **Directory Clue Collection (Quick Stops)**

**Implementation:**

**Multiple Directory Boards:**
- Directory boards likely at: Main Entrance, CambridgeSide Place Entrance, First Street Entrance, near escalators, Canal Park Entrance
- Each directory shows store listings with unit numbers (format: Level-Unit)
- Each has one clue (number, letter, word, symbol)
- Players can't stay long (security, crowds, time pressure)
- Need to collect all clues quickly
- Combine clues to form code/message

**Clue Types Using Actual Data:**

**Option A: Unit Number Patterns**
- Extract unit numbers from directory (e.g., 1-30, 2-11, 1-16)
- Pattern: Last digit of units, first digit of units, sum of digits
- Example: Apple (1-30) → 0, 1, or 3+0=3
- Multiple stores → Combine digits

**Option B: Store Name Letters**
- Extract letters from store names on directory
- Pattern: First letter, last letter, specific position
- Example: "APPLE" → A, E, or 3rd letter = P
- Multiple stores → Spell word

**Option C: Store Count Patterns**
- Count stores in specific categories
- Pattern: Number of stores starting with letter, number of food court eateries, etc.
- Example: How many stores start with "A"? (ALDO, APPLE, AT&T, ANOUSHELLA)
- Combine counts → Code

**Option D: Location-Based Clues**
- Clues based on directory location (which entrance, which floor)
- Pattern: Entrance name letters, floor number, section
- Example: "MAIN ENTRANCE" → Extract letters/numbers
- Combine location clues → Code

**Quick Stop Mechanics:**
- Telegram tells players which directory to check
- Players have 2-3 minutes at each
- Need to photograph/remember clue
- Move to next directory
- Combine all clues at end

**Actual Directory Locations to Use:**
- Main Entrance directory
- CambridgeSide Place Entrance directory
- First Street Entrance directory
- Canal Park Entrance directory
- Escalator area directory (Level 1 or Level 2)

**Story Integration:**
- Missing person left clues at each directory
- Guide character directs players to each
- Each clue is part of larger code
- All clues needed to solve puzzle

**Puzzle Types:**
- Number sequence (combine numbers)
- Letter extraction (spell word)
- Pattern recognition (identify pattern)
- Code breaking (decode message)

- Differences: Store order changed, unit numbers different, stores added/removed
