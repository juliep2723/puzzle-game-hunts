## Proposed Mechanics for Each Favorite Hook

*Each hook gets 5 mechanics mapped to Beats 2-6, following the difficulty progression: Beat 2 (repetitive/collection), Beat 3 (observation), Beat 4 (environmental), Beat 5 (logic/spatial), Beat 6 (integrated). Each mechanic involves both mall elements and mobile-browser elements.*

### Hook #1: Finding Shut-Off Symbols

**Beat 2 (Collection):** **Switch Symbol Scavenger Hunt**
- **Mall:** Find switch symbols (‚ö°, ‚öô, üîò) hidden in store signs, directory boards, floor tiles, or architectural details
- **Mobile:** Collect symbols into virtual "switch board" - each symbol has a number attached
- **Pattern:** Visit 8-10 locations, collect symbols + numbers, build sequence
- **Output:** First set of shut-off codes (e.g., "Switch 1 = 593")

**Beat 3 (Observation):** **Video Switch Sequence**
- **Mall:** Video shows Lex pointing at specific mall locations (escalator, elevator, directory)
- **Mobile:** Observe which switch symbols appear in video at each location
- **Pattern:** Match video locations to real mall locations, extract symbol sequence
- **Output:** Second set of shut-off codes + unlock next puzzle

**Beat 4 (Environmental):** **Directory Board Switch Extraction**
- **Mall:** Directory boards have switch symbols hidden in store listings (as icons, or extractable from store names)
- **Mobile:** Word search puzzle using store names - remaining letters reveal switch positions
- **Pattern:** Find stores with switch symbols nearby, extract letters from those store names
- **Output:** Third set of shut-off codes + concerning video about foam pressure building

**Beat 5 (Logic/Spatial):** **Switch Order Logic Puzzle**
- **Mall:** Find switch symbols on pipes, utility labels, ceiling fixtures (from ITEM 38 - pipes labeled "NATURAL GAS", "EXHAUST", etc.)
- **Mobile:** Logic grid puzzle - determine correct order of switches based on pipe flow direction, pressure zones, Rube Goldberg sequence
- **Pattern:** Match switch symbols to pipe labels, determine sequence based on flow
- **Output:** Complete switch sequence order + video showing Phantom's original plan

**Beat 6 (Integrated):** **Master Switch Panel Assembly**
- **Mall:** Final location (maybe CanalSide area or elevator lobby) - find actual switch panel location using all previous codes
- **Mobile:** Virtual switch panel - input all collected switch codes in correct order
- **Pattern:** Combine all 5 switch sequences into master code, enter in order
- **Output:** Master shut-off code unlocks final location/action
